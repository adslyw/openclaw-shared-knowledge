## 注意事项

- **幂等性**：`AppleCategorySyncService.sync` 已实现幂等（更新则不重复创建）
- **任务去重**：当前未实现，同一站点如果快速多次创建会生成多个任务（但数据最终一致）
- **失败重试**：`BACKGROUND_TASK_MAX_ATTEMPTS = 3`，失败时会自动重试
- **更新同步**：如需在更新 AppleSite 时也触发，修改 signals.py：将 `if created:` 移除或改为 `if created or instance.api_base_changed:`

## 回滚方案

如需恢复同步方式，删除 `tasks.py` 并恢复 signals.py 为：
```python
from django.db.models.signals import post_save
from django.dispatch import receiver
from .models import AppleSite
from .services.category_sync import AppleCategorySyncService

@receiver(post_save, sender=AppleSite)
def sync_apple_categories(sender, instance, created, **kwargs):
    if created:
        AppleCategorySyncService.sync(instance)
```
