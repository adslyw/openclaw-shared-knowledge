## 🔄 第二阶段优化 (21:26-22:55 CST)

### 主要变更
1. **风格恢复**: 从 JW Player 深色主题回退到简洁 Tailwind 默认风格
2. **分组展开/收起**: M3U 分组可点击展开/收起子频道列表
3. **分批加载**: 大型 M3U 源每次只加载200个频道，避免DOM过载
4. **海报显示**: 解析 M3U 的 `tvg-logo` 属性并显示为缩略图
5. **频道项高度优化**: 从 40px 增加到 80px+，海报更大更清晰
6. **侧边栏收起修复**: 用 CSS 变量控制宽度，收起时隐藏文字只显示海报
7. **Tooltip 支持**: 收起状态下 hover 海报显示频道名称

### 数据结构变更
- M3U 源新增 `expanded` 字段（默认 true）
- M3U 源新增 `visibleCount` 字段（默认 200）
- 子频道不再作为独立项添加到 channels 数组
- 旧数据自动迁移清理

### 修复的问题
- `showAddModal` 方法丢失（被误删）
- 展开/收起状态未持久化
- 旧格式子频道导致计数重复
- 侧边栏宽度被 Tailwind 类名覆盖无法收起
- 频道项高度过小导致文字被挤压

### 关键教训
- **CSS 变量 > Tailwind 类名**: 当需要动态切换状态时，CSS 变量更可靠
- **分批渲染**: 大型列表必须分批加载，否则 DOM 节点过多导致卡顿
- **数据迁移**: 修改数据结构时必须考虑旧数据的兼容性
