# Menu Organizer Skill

## 描述
将原始食谱文本整理成标准的4列Markdown表格（日期、早餐、午餐、晚餐），并按预设规则分类显示。

## 规则

### 1. 表格结构
- **4列**：日期 | 早餐 | 午餐 | 晚餐
- **行数**：基于输入的天数（通常为周一至周五，周末可空着或删除）

### 2. 分类体系（5类）
固定显示顺序：**主食 → 配菜 → 面点 → 副食 → 汤品**

每个菜品必须归入一个类别，归类规则：

| 类别 | 包含内容 | 示例 |
|------|---------|------|
| **主食** | 粥、糊汤、稀饭、米饭（午餐自动补充）、面条类（刀削面、扯面、方便面、炸酱面等） | 胡辣汤、红豆糊汤、红薯稀饭、紫米稀饭、肉丝炒刀削面、酸菜方便面、西红柿油泼扯面 |
| **配菜** | 炒菜、炖菜、凉拌菜、素拼、时蔬等 | 杏鲍菇牛柳、芹菜鸡丁、炝炒菜花、老干妈土豆丝、凉拌黄瓜、素拼凉菜 |
| **面点** | 饼类、包子、火烧、煎包、擀面皮、油饼、茄子包等面食制品 | 油饼、酱香饼、水煎包、肉火烧、擀面皮 |
| **副食** | 蒸制食品、卤蛋、卤肉类、洋芋叉叉、**糕点类**（甑糕、年糕、枣糕等） | 蒸紫薯、蒸鸡蛋、蒸玉米、卤鸡蛋、洋芋叉叉、甑糕、年糕 |
| **汤品** | 各类汤、羹、炖品（冰糖雪梨等） | 紫菜蛋花汤、粟米羹、面汤、冰糖雪梨、冬瓜汤 |

**特殊情况：**
- 早餐通常无汤品，胡辣汤/糊汤归主食
- **午餐主食为米饭时，必须搭配汤品**（如萝卜丝汤、冬瓜虾米汤、紫菜蛋花汤等）
- **午餐主食为米饭且配菜包含鱼类（如红烧鱼块）时，汤品应选择鱼汤**（如鱼汤、豆腐鱼汤等）
- 晚餐主食通常是稀饭或面条
- **糕点类**（甑糕、年糕、枣糕等）必须归为**副食**，即使它们由米制成

### 3. 显示规则
- 午餐主食：**缺失自动补"米饭"**（唯一自动补充项）
- 其他类别：**空缺不显示**（不显示"无"或空行）
- 每类别一行显示，格式：`**类别**：菜品列表`（例如 `**主食**：小米红枣稀饭、青笋木耳山药`）
- 同一单元格内不同类别用 `<br>` 分隔（在 Markdown 源码中）
- 表格内左对齐，自动换行，垂直居中

**图片渲染说明**：
- Markdown 源码中的 `**`、`<br>`、`：` 等标记在生成图片时会**完全清除**
- 渲染效果：类别名**加粗**显示，紧接着冒号后的菜品列表（同一行）
- 长菜品列表会自动换行，保持可读性

### 4. 输出与保存
- **Markdown 表格**：用于文档记录（保存为 `~/FoodMenue/YYYY-MM-DD.md`）
- **PNG 图片**：用于快速查看（保存为 `~/FoodMenue/YYYY-MM-DD-table.png`）
- **图片规格**：
  - 字体：NotoSansCJK-Bold，20号（已增大两号）
  - 表头行高：50px，内容行高自适应
  - **自动换行**：单元格内容超过列宽时自动换行
  - **垂直居中**：内容在单元格内精确垂直居中
  - **自适应列宽**：根据内容最大宽度调整列宽（基础宽度：日期=80px，早餐=380px，午餐=380px，晚餐=360px）
  - 尺寸：约 1240×700px（根据内容行数调整）
  - 背景：表头绿色（#4CAF50）+ 白字，内容区白色 + 黑字

### 5. 输入格式
文本输入，每日期一行或多行，菜品用句号、顿号、逗号、空格或换行分隔。

示例：
```
周一 胡辣汤。油饼。卤鸡蛋。蒸紫薯。杏鲍菇牛柳。芹菜鸡丁。...
```

## 使用
调用 skill 时传入原始食谱文本，返回整理好的 Markdown 表格，并自动生成图片。

## 示例输入 & 输出

**输入（原始文本）：**
```
周一 小米红枣稀饭。青笋木耳山药。酸辣白菜。蒸鸡蛋，茴香煎饼。
周二 豆腐脑。菜夹馍。蒸鸡蛋。
周三 花生红枣稀饭。柞水小炒。青椒豆干。粉条包子。
...
```

**输出（Markdown 表格源码）：**
| 日期 | 早餐 | 午餐 | 晚餐 |
|------|------|------|------|
| 周一 | **主食**：小米红枣稀饭<br>**配菜**：青笋木耳山药、酸辣白菜<br>**副食**：蒸鸡蛋<br>**面点**：茴香煎饼 | **主食**：米饭<br>**配菜**：菌菇炒肉、肉末茄子、西红柿鸡蛋、杏仁拌菠菜<br>**汤品**：枸杞醪糟汤 | **主食**：火腿炒方便面<br>**汤品**：西湖牛肉羹 |
| 周二 | **主食**：豆腐脑<br>**面点**：菜夹馍<br>**副食**：蒸鸡蛋 | **主食**：红烧牛肉面<br>**汤品**：肉汤 | **主食**：红豆稀饭<br>**配菜**：炝炒三丝儿、干煸豆角<br>**面点**：韭菜盒子 |
| ... | ... | ... | ... |

**图片渲染效果：**
- 类别名（如"主食"）显示为**加粗**
- 类别名后紧跟**中文冒号**和菜品列表（顿号分隔）
- 每个类别一行，不显示 `**` 和 `<br>` 源码标记
- 示例（周一早餐单元格）：
  ```
  主食：小米红枣稀饭、青笋木耳山药、酸辣白菜
  副食：蒸鸡蛋
  面点：茴香煎饼
  ```
